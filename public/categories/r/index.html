Logistic2

April 08, 2019
R
Regresión Logistica Multiple. Consideremos ahora el esceario de la predicción de una variable binaria a partir de un conjunto de variables independientes. El modelo es: \[log\bigg(\frac{P(X)}{1-P(X)}\bigg)= \beta_0 +\beta_1 X+...+\beta_p X_p\] Donde \(X=(X_1...,X_p)\) son p variables explicativas, independientes o regresores. Podemos reescribir la ecuación anterior como: \[P(X)=\frac{e^{\beta_0 +\beta_1 X+...+\beta_p X_p}}{1+e^{\beta_0 +\beta_1 X+...+\beta_p X_p}}\] Usando el método de máxima verosimilitud estimamos los coeficientes del modelo log(odds) que nos permiten posteriormente determinar las probabilidades asociadas.

PCA

January 28, 2019
R
Análisis de Componentes Principales (PCA). Motivación El método PCA permite reexpresar un conjunto multidimensional de datos (ej. k dimensiones) que contiene variables correlacionadas ( ej. que aportan información redundante), en un subconjnto de datos de menor dimensión. Con PCA respondemos preguntas como: ¿Qué variables estan correlacionadas? ¿Podemos representar los datos de una manera más clara ej.mediante un menor número de dimensiones? ¿Qué variables tienen la mayor influencia para explicar la variación de los datos?

RL-Multiple

January 26, 2019
R
Regresión Multiple. La estimación de modelos bajo esta estructura permite incorporar p variables independientes, lo que contribuye para explicar una mayor parte de la variación en \(Y\) al controla por determiantes de interés. \[\begin{align} Y=\beta_0+\beta_1 X_1+\beta_2 X_2+...+\beta_p X_p+\epsilon \tag{1} \end{align}\] Donde \(\beta_j\) indica el efecto promedio sobre Y de un cambio de una unidad en \(X_j\) manteniendo los otros parámetros constantes. Esta estructura del modelo nos permite controlar por \(X_j\) variables explicativas de Y.

Parámetros estimados y su precisión. ¿Qué tan preciso es un parámetro estimado mediante el modelo de regresión lineal? Una manera de medir su precisión es mediante el cálculo del error estándar SE este nos indica el monto promedio en el que el parámetro \(\hat \beta\) estimado difiere del valor poblacional \(\beta\) Note que nos interesa determinar la precisión del parámetro muestral con relación al poblacional en este caso tomamos el ejemplo de la estimación de media muestral \(\hat \mu\) con relación a su correspondiente valor poblacional \(\mu\).

RL-Modelo

January 15, 2019
R
Formalización del modelo En términos matemáticos podemos describir la relación entre las variables del modelo con la expresión siguiente: \[\begin{align} Y \approx \beta_0 + \beta_1 X \tag{1} \end{align}\] Una vez que hemos estimado los parámetros del modelo podemos usarlos para hacer la predicción de \(\hat y\) \[\begin{align} \hat y= \hat \beta_0 + \hat \beta_1 x \tag{2} \end{align}\] Note que la diferencia entre la linea de regresión y cada observación (lineas grises) da cuenta del error de predicción.
